# xml2json

> Convert XML from clipboard or current document to JSON, converted data can be use for [xmlbuilder](https://www.npmjs.com/package/xmlbuilder) or [xml2js](https://npmjs.com/package/xml2js)

Are you working with XML builder tools day by day? xml2json can help you create the final object, just edit the variables that need dynamic data instead of typing each object key, attribute and no typos mistake.

## Install
Press `F1` and type `ext install xml2json`

## Usage
- Invoke command pallate (Cmd/Ctrl + Shift + P)
- Type `xml to json`, choose expected format, hit enter!

**Enjoy!**
